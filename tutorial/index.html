How to obtain Aim gods account token

This tutorial will not work for people on mobile

Step 1:

Go to https://aimgods.finalmouse.com and login

Step 2:

Press F12 or Right click and click Inspect to open developer tools

Step 3:

When the dev tools window opens click on the application tab

Step 4:

Expand the Local storage option then click on the https://aimgods. link

Step 5:

Click on the ag_tokens field then copy everything in between the quotations in the id field of the window that gets opened, this is your authorization token

The id token expires after an hour but should not be shared with anyone. This website does not store any tokens you input

If you don't believe me and want to look for yourself the source code is available at https://github.com/Slendy/GoldenKeyOpener
